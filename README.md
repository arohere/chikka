# Kartus, a bot for the VITC discord server

### Requirements:

- python

### Getting started

Add python to PATH if you haven't already

Powershell

```
python -m venv .venv
.venv/Scripts/Activate.ps1
pip install -r requirements.txt
```

CMD

```
python -m venv .venv
.venv/Scripts/activate.bat
pip install -r requirements.txt
```

Once the changes are done

```
python deps.py
```
